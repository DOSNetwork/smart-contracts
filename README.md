### Dev:
- `$ git clone https://github.com/DOSNetwork/eth-contracts`
- `$ npm install`
- `$ npm -g install truffle`
- `$ npm install -g ganache-cli` or installing its [graphic interface](https://truffleframework.com/ganache)

### Compile:
- `$ truffle compile`

### Deploy:
- `$ truffle migrate --reset`

### Test:
- `$ truffle test`
